## このゲームの概要
### 基本ルール
- あなた（プレイヤ）とコンピュータ（ディーラ）の1対1の対戦です。
- カードの合計値を21に近づけること。ただし、21を超えると「バースト」となり、その場で負けが決定します。

### カードの数え方
- 2〜10: そのままの数字として数えます。
- J・Q・K (11〜13): すべて 「10」 として数えます。
- A (1): このゲームでは 「1」 として計算されます。（←本来のルールとは違います）

### ゲームの流れ

#### *1.* スタート:
 
 スペースキーを押すと、プレイヤに2枚（表）、ディーラに2枚（1枚は裏、1枚は表）配られます。
 
#### *2.* プレイヤのターン: 
↳ヒット: 「ヒット」ボタンを押してカードを1枚追加します。21を超えない限り5枚までカードを追加できます。

↳スタンド: 「スタンド」ボタンを押すと、自分の合計値を確定させ、ディーラのターンへ移ります。
 
#### *3.* ディーラのターン:
　
 あなたがスタンドした瞬間に、裏向きだったディーラの2枚目のカードが公開されます。
 
#### *4.* 勝敗
- プレイヤーが21を超えた⇒その時点で負け（バースト）
- ディーラーが21を超えた⇒プレイヤーの勝ち
- プレイヤー ＞ ディーラー⇒プレイヤーの勝ち
- プレイヤー ＜ ディーラー⇒ディーラーの勝ち
- プレイヤー ＝ ディーラー⇒引き分け
